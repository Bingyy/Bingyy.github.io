杀死一只知更鸟观后感

发表于 2018-03-01 | 分类于 电影评论

这是第一篇电影观后感。

以往看了很多电影，豆瓣Top1000大概有十分之四都看过了。但是看过就看过了，只是当时觉得舒服了，过瘾了而已，并未有任何自己的输出内容。而我一直在强调，只有输出才能真的锻炼你的思维。对于学习专业知识而言，我倒是做了许多，但是电影，音乐，文学作品等都没有在输出上用力。

本篇是写于2018.3.1号，晚间看完《杀死一只知更鸟》电影之后的感触。

电影是1963年的黑白片。

不妨从电影中头尾相接的一句话说起：

“It is a sin to kill a mockingbird.”
杀死一只知更鸟是种罪恶。

老实说，这本书被推荐很久了，记得大四时候，和一个小伙伴在寝室阳台聊天谈起这本书，我是没有读过的，他很推崇这本书，给我讲述了这本书的大致情节。但是这种听来的东西，自己的感受会很轻，因为一本好书或者一本好电影，甚至放宽至任何作品，第一手的感受和别人传递过来的是不太一样的。Mark下了这本书，并很快在Kindle商店买到了这本小说，但是放着一直没太看。另一本感觉蛮贵的博拉尼奥的《2666》，也是买了放在Kindle中，还没看完序言。惭愧。

我觉得理解了杀死一只知更鸟这句话的含义，整个电影的哲学思考，或者说社会意义也就很明确了。

知更鸟是什么鸟，我不是很清楚，但根据电影所说，这种鸟除了给人类唱歌，什么坏事也没有干过。不像麻雀，农民会觉得这种鸟偷吃粮食，从人类视角看来，会有些许罪恶。

那么对于知更鸟这种，毫无罪恶甚至还给人类带来审美享受，杀死一只知更鸟，人类的罪恶便昭然若揭。

影片显然不是在讲述这种直白的意向，一定意有所指，隐喻的手法。

我读了一小部分书籍，只看到书中三个小孩子Jem, Scout, Dill的日常探险，以及风传邻居是个疯子，是个危险因素，他们都不敢接近，并以能走近邻居的房子为勇气的体现。这个邻居叫Arthur Boo Radley。之所以提出名字，是因为这样前期塑造的一个危险的有精神病的人，到后面恰恰是点睛之笔。

本部电影的核心剧情在于Jem,Scout的爸爸Atticus Finch为黑人Tom Robinson打官司。

Atticus Finch是个白人律师，谈吐儒雅，育儿有术。不由得让我想起美剧《This is us》中的Jack,总是能够平等地和自己孩子讲道理，传递理性与善良。Atticus也是一样，当孩子们耍脾气或者做错了事情，并不是发火说气话或者暴力，而是坐下来，给孩子讲道理。让孩子明白背后的道理，通常孩子都会明白。这让我感到小朋友实际上要比我们想象的成熟的多，也明事理的多。

Tom是个左手残废的黑人年轻小哥，也就是句子中的一只“知更鸟”。被镇子上的白人穷鬼Bob Ewell以及他的女儿Mayella诬陷强奸了Mayella。最后官司失败，Tom在转进监狱时逃跑被错手枪杀。

这个剧细节很多，书上的内容应该要更详细，电影进行了精选。

Mayella作为一个白人女性，主动诱惑Tom，但未得逞而恼羞成怒，被酗酒的父亲暴打却诬赖Tom。为了转移自己的羞愧，不惜以一条人命为代价。

而在法庭上，Atticus抽丝剥茧，将事实演绎出来，明明大家都知道了黑人Tom无罪，但是陪审团仍然判定Tom有罪。这让人不由得怀疑陪审团这种制度。但是又想起《十二怒汉》，也是这种陪审团，却能还以事实真相。可能考虑问题更得注意历史进程。毕竟那个黑人还被白人歧视的年代。

整部影片到黑人Tom死亡以后，已经完整了。诬陷别人的Bob Ewell得逞，Atticus除了痛心疾首以外，也并无其他什么动作。剧中的两只知更鸟已经显现，Boo明明没有做什么，只是在剧中被当做危险因素。Tom明明没错，却被判有罪而无辜死去。恶人该收手了吧，但是没有。

Bob Ewell企图报复律师，想伤害Jem和Scout，幸好Boo Radley出现杀死了Bob Ewell.

好一个苍天放过谁。但纵看整个剧情，其实并不是正义战胜邪恶，而是邪恶在霸凌正义。

而正义又当如何定义，天知道。

发表于 2018-02-26 | 分类于 区块链

Solidity语言简介

@(区块链)

本文笔记自维基。

https://en.wikipedia.org/wiki/Solidity

Solidity是面向智能合约的编程语言。也即Solidity发明出来的目的就是为了实现智能合约。但是请注意，智能合约的实现手段不止是Solidity，其他的语言比如Go, JS等同样可以用来实现智能合约。只不过Solidity专门用来做这件事情，是不是就比其他的语言更好用，需要慢慢发现。

Solidity的开发人员是几个前以太坊平台核心开发者，目的是为了方便大家在以太坊平台上构建智能合约。

本篇笔记大纲：

  1. 历史
  2. 描述
  3. 开发平台的可用性
  4. 区块链平台

历史

Solidity最初提出来是在2014年8月份，由Gavin Wood提出。后来这个语言被以太坊团队接手，是四大针对以太坊虚拟机EVM制作的语言之一。四大语言分别是Serpent, LLL, Viper(实验性)，Mutan(废弃)。

当前，Solidity是以太坊平台的是首选语言。此外，Solidity还是其他的与以太坊竞争的私有区块链平台上的首选开发语言，比如Monax以及Hyperledger.

描述

Solidity是静态类型语言，运行在EVM上。Solidity会被编译成字节码，然后在EVM上执行。这个可以类比Java语言的执行，用的是JVM。Solidity将智能合约的编写变得更加容易。

Solidity的语法是围绕着ECMAScript设计的，因此对于Web开发者而言，将会非常舒服。

开发平台

  • Remix，官方的IDE
  • Visual Studio
  • ConsenSys Enterprise
  • Tendermint on Microsoft Azure by PWC
  • ErisDB by AWS

区块链平台

Solidity支持如下几个平台：

  • Ethereum
  • Tendermint & ErisDB
  • Zeppelin by Digital Currency Group
  • Counterparty(运行在Bitcoin网络上)

分布式自治组织

发表于 2018-02-26 | 分类于 区块链

分布式自治组织 : DAO

@(区块链)

https://en.wikipedia.org/wiki/Decentralized_autonomous_organization

引出DAO的概念之前，需要澄清与The DAO的关系，DAO表示的是一种组织形式，而The DAO则是具体的一个DAO形式的公司。

DAO全称是Decentralized Autonomous Organization. 也称作Decentralized Autonomous Corporation. 这种形式的组织的运行完全通过计算机程序，这类程序称作智能合约。 而一个DAO组织的财务事务记录以及程序规则都是放在区块链上运行的。 这类组织的精准的法律状态是不清楚的。

这种完全由计算机代码控制运作的类似公司的实体，没有founder， 没有CEO，没有CTO，没有人事财务研发市场销售部，彻底根除了影响公司效率的认为因素，对于现行的公司体制是历史性的变革。

The DAO

基于以太坊的众筹项目。

通过以太坊的智能合约功能，很容易开发出定制化的众筹代码。The DAO的众筹代码仅仅只有几百行。The DAO的成功，启发了后来井喷式的众筹项目，尤其是区块链领域的，几乎都在以太坊平台进行。

The DAO的目标是同时为商业的和非商业的企业提供去中心化的商业模式。

The DAO是无国籍的，不绑定任何国家，因此政府如何应对这种无国籍的基金还不可知。

The DAO在2016年进行众筹。是史上最大笔的众筹。使用的是以太坊的智能合约平台完成。

The DAO本质上是个风险投资基金(VC)。通过以太坊筹集资金，这些资金会被锁定在智能合约中，行为由智能合约中的代码主导。

每个众筹人按照出资数额获取相应的DAO代币，具有审查项目和投票表决的权利。

从这个意义上看，DAO代币像是股票。参与众筹的人还有权提出投资项目的议案，供The DAO审核。投资议案（注意到The DAO是VC）由全体代币持有人投票表决。每个代币代表一票。议案得到需要的足够的票数支持，相应的款项就会自动划给该投资项目。

传统的VC是由经验丰富老道的专业人士制定，而在The DAO中，决策来自于众智。即：综合许多人的智慧，可以得出比某个专家更好的结论。

P.S. 这一点我是持保留态度的。

投资收益按照一定的规则回馈给代币持有人。

道氏理论序言

@(道氏理论)

个股价格波动的背后，总隐藏着市场整体趋势的变动。

《一个理论的演变》

道氏理论这个名称是道琼斯的好朋友尼尔森在《股票投机原理》中提出的。

平均价格指数。

一个重要假设：历史将会不断地重演。

如果初学者能够自己绘制价格走势图，将极大增强自己运用图标研究价格走势的能力。

道氏理论的思想完全是针对平均价格指数的分析。

将道氏理论发扬广大的是汉密尔顿。

再次强调，平均价格指数是一个有效的股市预测方法。

作用，反作用与相互作用

任何公开交易的股市，均存在这三种作用：

  • 作用
  • 反作用
  • 相互作用

道氏理论以实效性作为其理论基础。

人类的本性：经济的繁荣驱使人们大胆冒进，对冒进行为的后悔，又伴随着经济大萧条。

这在我玩币的过程中，感受非常明显，市场的踩踏行为是非常可怕的，感性的作用要远远大于理性的作用。牛市来临时，人人都是股神，熊市到来时，一片哀嚎。
